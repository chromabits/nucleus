Get the default attributes for this node.

Get the default attributes for this node.

Usually basic nodes such as won't have any defaults. However, if you are using a CSS framework, it might be useful to have default classes on more complex classes.

Example: On Bootstrap, Tables need to have class='table'. Instead of writing new Table(['class' => 'table'], [...]), we can extend the Table class and override the getDefaultAttributes to return ['class' => 'table']
